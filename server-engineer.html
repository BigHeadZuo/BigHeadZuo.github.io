🐧 阶段一：系统基础（0-4 个月）

精通 Linux/Windows 系统管理，打好运维基本功

1.1 Linux 系统管理
⏱ 60 学时
🎯 精通 Linux 系统管理，能独立运维生产服务器
核心知识点
  • Linux 发行版选择（CentOS/Rocky/Ubuntu/Debian/RHEL）
  • 文件系统层级（FHS）与目录结构
  • 用户/组管理、权限管理（chmod/chown/ACL/SUID/SGID/Sticky）
  • 软件包管理（yum/dnf/apt/rpm/dpkg）
  • systemd 服务管理（unit 文件编写/依赖管理）
  • 进程管理（ps/top/htop/kill/nice/cgroups）
  • 磁盘管理（fdisk/parted/LVM/RAID/fstab）
  • 网络配置（ip/ss/nmcli/NetworkManager/netplan）
  • SSH 配置与密钥管理（跳板机/ProxyJump）
  • SELinux/AppArmor 安全模块
  • 内核参数调优（sysctl）
推荐教程
  • Linux Journey（交互式学习）
  • 兄弟连 Linux 教程（B站）
  • RHEL 9 官方文档
  • Linux From Scratch
  • 书籍：《鸟哥的 Linux 私房菜》《UNIX/Linux 系统管理技术手册》
🔧 实战练习

① 安装 Rocky Linux 9，配置网络/用户/SSH 密钥登录
② 配置 LVM 逻辑卷并在线扩容
③ 编写 systemd service 文件管理自定义服务
④ 完成 OverTheWire Bandit 全部关卡
