🌱 阶段一：基础筑基（0-6 个月）

打好计算机科学基础，掌握核心编程与系统管理能力

1.1 计算机组成与操作系统
⏱ 30 学时
🎯 理解计算机硬件架构、操作系统原理，为后续安全研究打下底层基础
核心知识点
  • CPU 架构（x86/ARM）、寄存器、指令集、内存层次结构
  • 进程与线程管理、进程间通信（IPC）、调度算法
  • 虚拟内存、分页机制、内存保护（DEP/ASLR 原理）
  • 文件系统（ext4/NTFS）、磁盘 I/O、RAID
  • 系统调用、中断机制、内核态与用户态
  • 启动流程（BIOS/UEFI → Bootloader → Kernel）
推荐教程
  • 王道考研 - 操作系统（B站）
  • 北大操作系统 - Coursera
  • OSTEP 操作系统导论（免费教材）
  • Nand2Tetris - 从零构建计算机
🔧 实战练习

① 在 VirtualBox 中安装 Ubuntu/Kali，熟悉双系统环境
② 使用 top/htop/ps 观察进程，用 strace 跟踪系统调用
③ 完成 Nand2Tetris 前 6 章项目

1.2 计算机网络核心
⏱ 40 学时
🎯 精通 TCP/IP 协议栈，能用 Wireshark 分析任意网络流量
核心知识点
  • OSI 七层模型 vs TCP/IP 四层模型
  • 以太网帧、ARP 协议、MAC 地址表
  • IP 协议、子网划分（CIDR）、路由原理（RIP/OSPF/BGP）
  • TCP 三次握手/四次挥手、滑动窗口、拥塞控制
  • UDP 协议、DNS 解析流程（递归/迭代）
  • HTTP/HTTPS 协议、TLS 握手、证书链
  • DHCP、NAT、VPN 原理（IPSec/WireGuard）
推荐教程
  • 韩立刚 - 计算机网络（B站）
  • Google IT Support - Coursera
  • Beej's Guide to Network Programming
  • 书籍：《TCP/IP 详解 卷1》《图解 HTTP》
🔧 实战练习

① 用 Wireshark 抓包分析 TCP 握手、DNS 查询、HTTPS 流量
② 用 Python socket 编写简易 HTTP 服务器和客户端
③ 搭建家庭实验网络：配置 VLAN、静态路由、端口转发

⏱ 50 学时

  • 用户与组管理、PAM 认证、sudo 配置
  • 包管理（apt/yum/pacman）、源码编译安装
  • 服务管理（systemd/systemctl）、日志系统（journalctl/rsyslog）
  • 网络配置（ip/ss/iptables/nftables/firewalld）
  • SSH 配置与密钥管理、远程管理
  • 磁盘管理（fdisk/LVM/RAID）、定时任务（cron）
