阶段一

🌱 基础筑基（0-6个月）

计算机基础
操作系统原理（Linux/Windows/macOS）、计算机网络（TCP/IP、OSI七层模型、三次握手/四次挥手）、数据结构与算法基础、进程与线程、内存管理
硬件与连接类型
计算机硬件组件认知（CPU/内存/硬盘/网卡）、连接类型与功能（NFC/WiFi/蓝牙/红外线）、网络设备（路由器/交换机/AP/防火墙）、接口类型（RJ45/光纤/USB）
编程语言入门
Python（首选，用于自动化脚本和工具开发）、Bash Shell 脚本（日常运维必备）、PowerShell（Windows 环境管理）、SQL 基础语法（数据库查询与管理）、正则表达式
Linux 运维基础
常用命令（grep/awk/sed/find）、文件权限与 ACL、用户与组管理、Systemd 服务管理、Crontab 定时任务、日志分析（/var/log）、SSH 远程管理、各发行版差异（Ubuntu/CentOS/Debian）
网络基础实操
Wireshark 抓包分析、Nmap 端口扫描与服务识别、DNS/HTTP/HTTPS/FTP/SFTP/SMTP/NTP 协议理解、子网划分（CIDR/子网掩码/默认网关）、公网IP vs 私网IP、localhost/loopback
网络术语与概念
VLAN、DMZ、ARP、VM、DHCP、DNS、NAT、IP、VPN、MAN/LAN/WAN/WLAN、NAS 与 SAN 基础、SSL/TLS 基础概念
网络拓扑与排障工具
网络拓扑类型（星型/环型/网状/总线型）、排障工具实操（ping/tracert/nslookup/ipconfig/netstat/arp/dig/route）、端口扫描器、协议分析器、iptables 基础
虚拟化与实验环境
VMware/VirtualBox 虚拟机搭建、Proxmox/ESXi 了解、Hypervisor 概念（Type1/Type2）、GuestOS vs HostOS、Docker 基础使用、搭建个人实验网络环境
📚 推荐资源
《鸟哥的Linux私房菜》、《计算机网络：自顶向下方法》、《Python编程：从入门到实践》、B站韩立刚老师网络课程
🎯 阶段目标
能独立搭建 Linux 服务器、用 Python 写简单自动化脚本、理解网络数据包流转过程、能用 Wireshark 分析基本流量
📜 建议认证
CompTIA Network+ / 华为 HCIA / RHCSA（红帽认证系统管理员）
阶段二

🔒 安全入门（6-12个月）

Web 安全基础
OWASP Top 10 深入理解、SQL注入（联合注入/盲注/堆叠注入）、XSS（反射型/存储型/DOM型）、CSRF、文件上传漏洞、SSRF、XXE、命令注入、目录遍历
渗透测试入门
Burp Suite 全功能使用（Proxy/Repeater/Intruder/Scanner）、SQLMap 自动化注入、Metasploit 基础模块使用、DirBuster/Gobuster 目录扫描
密码学基础
对称加密（AES/DES）、非对称加密（RSA/ECC）、哈希算法（MD5/SHA）、数字签名与证书、PKI 体系、SSL/TLS 握手流程、密码破解原理、Salting 与 Key Exchange、混淆技术（Obfuscation）
认证方法论
Kerberos 认证协议、LDAP 目录服务、SSO 单点登录、RADIUS 认证、本地认证机制、证书认证、MFA 多因素认证与 2FA、Authentication vs Authorization 区别
安全协议对比
FTP vs SFTP、SSL vs TLS、HTTP vs HTTPS、IPSec、DNSSEC、LDAPS、SRTP、S/MIME、安全协议与非安全协议的选择原则
端点安全基础
Antivirus vs Antimalware、EDR 端点检测与响应、DLP 数据防泄漏、ACL 访问控制列表、防火墙与下一代防火墙（NGFW）、HIPS 主机入侵防御、沙箱（Sandboxing）
无线安全
WPA vs WPA2 vs WPA3 vs WEP 对比、EAP vs PEAP 认证、WPS 安全风险、无线网络攻击（Deauth/Evil Twin/Rogue AP）
安全工具使用
Kali Linux 环境搭建与工具集、Nessus 漏洞扫描与报告、Hydra/Medusa 暴力破解、John the Ripper 密码破解、Hashcat GPU 破解
靶场实战练习
DVWA 全等级通关、Pikachu 靶场、WebGoat、sqli-labs、upload-labs、XSS Challenge、TryHackMe 入门路径
信息安全法规
《网络安全法》、《数据安全法》、《个人信息保护法》、等级保护制度基础、渗透测试授权与法律边界
📚 推荐资源
《Web安全攻防：渗透测试实战指南》、《白帽子讲Web安全》、PortSwigger Web Academy（免费在线靶场）、先知社区、FreeBuf
🎯 阶段目标
能独立完成基础 Web 渗透测试、编写简单漏洞利用脚本、理解常见漏洞原理与修复方案、能输出规范的渗透测试报告
📜 建议认证
CompTIA Security+ / CISP-PTE / eJPT（eLearnSecurity）
阶段三

⚔️ 渗透测试进阶（1-2年）

高级渗透技术
内网渗透（信息收集/代理转发/端口转发）、域渗透（Kerberoasting/AS-REP Roasting/DCSync）、权限提升（Windows提权/Linux SUID/内核漏洞）、横向移动（PsExec/WMI/DCOM/WinRM）
隧道与代理技术
frp/Chisel/Neo-reGeorg 内网穿透、SSH 隧道、DNS 隧道、ICMP 隧道、Socks5 代理链、流量加密与隐蔽通信
代码审计
PHP 代码审计（ThinkPHP/Laravel 框架漏洞）、Java 反序列化漏洞（Fastjson/Shiro/Log4j）、Python 代码审计、.NET 安全审计、自动化审计工具（Semgrep/CodeQL）
红队技术
Cobalt Strike Beacon 操作、免杀技术（Shellcode加载器/混淆/加壳）、钓鱼攻击（邮件/网站克隆/Office宏）、社会工程学、APT 攻击链模拟、C2 基础设施搭建、LOLBAS/GTFOBins（利用合法工具进行攻击）
常见攻击类型详解
社工类：Phishing/Vishing/Whaling/Smishing/Spam vs Spim/Shoulder Surfing/Dumpster Diving/Tailgating/Impersonation；网络类：DoS vs DDoS/MITM/Spoofing/DNS Poisoning/ARP欺骗/VLAN Hopping/Replay Attack/Pass the Hash
Web 攻击与恶意软件
Buffer Overflow/Memory Leak/XSS/SQL Injection/CSRF/Directory Traversal/Watering Hole/Drive-by Attack/Typo Squatting；恶意软件类型：病毒/蠕虫/木马/勒索软件/Rootkit/Keylogger/Botnet/Fileless Malware
漏洞赏金实战
参与 SRC 漏洞提交（各大厂商安全应急响应中心）、HackerOne/Bugcrowd/补天平台实战、编写高质量漏洞报告、漏洞复现与 PoC 编写
移动安全基础
Android 逆向（APK反编译/Smali/Frida Hook）、iOS 安全基础、移动应用渗透测试、API 安全测试、小程序安全
📚 推荐资源
《内网安全攻防》、《Web安全深度剖析》、Hack The Box / VulnHub 靶机、红队知识星球、先知社区高级文章
🎯 阶段目标
能独立完成内网渗透全流程、具备代码审计发现漏洞的能力、在 SRC 平台有漏洞提交记录、能编写自动化渗透脚本
📜 建议认证
OSCP（必考）/ CEH / CISP-PTS / CRTO（红队认证）
阶段四

🛡️ 安全防御体系（2-3年）

安全运营（SOC）
SIEM 平台部署与运营（Splunk/ELK/QRadar）、安全事件分析与分级、告警研判与误报处理、应急响应流程（遏制/根除/恢复）、安全运营指标（MTTD/MTTR）
防御技术
WAF 配置与规则编写（ModSecurity/云WAF）、IDS/IPS 部署（Suricata/Snort）、防火墙策略设计（iptables/pf）、蜜罐部署（HFish/T-Pot）、零信任架构设计、DLP 数据防泄漏
安全开发（DevSecOps）
SAST 工具集成（SonarQube/Semgrep）、DAST 工具（OWASP ZAP）、SCA 依赖检查（Snyk/Dependabot）、CI/CD 安全流水线、容器安全（Docker 镜像扫描/K8s RBAC/Pod安全策略）
威胁情报
MITRE ATT&CK 框架深入应用、威胁狩猎（Threat Hunting）方法论、IOC/TTP 分析、威胁情报平台（MISP/OpenCTI）、暗网情报收集
应急响应与取证
Windows/Linux 入侵排查、Webshell 查杀、恶意进程分析、日志溯源、内存取证（Volatility）、磁盘取证（Autopsy）、攻击时间线还原
事件响应六步法
① Preparation 准备 → ② Identification 识别 → ③ Containment 遏制 → ④ Eradication 根除 → ⑤ Recovery 恢复 → ⑥ Lessons Learned 总结复盘；IR 工具：nmap/curl/hping/wireshark/FTK Imager/memdump/winhex
日志类型与分析
Event Logs（Windows事件日志）、Syslog（Linux系统日志）、Netflow（网络流量日志）、Packet Captures（数据包捕获）、Firewall Logs（防火墙日志）、日志收集与关联分析
系统加固概念
MAC-based/NAC-based 访问控制、端口阻断（Port Blocking）、组策略（Group Policy）、ACL 访问控制列表、Sinkhole 技术、补丁管理（Patching）、跳板机（Jump Server）、操作系统加固基线
威胁分类与情报工具
Zero Day/Known vs Unknown 威胁/APT 高级持续威胁；蓝/红/紫队职责；False Positive vs False Negative/True Positive vs True Negative；常用工具：VirusTotal/Joe Sandbox/Any.Run/URLVoid/URLScan/WHOIS
安全框架与标准
MITRE ATT&CK 框架、Cyber Kill Chain 杀伤链、Diamond Model 钻石模型；标准体系：NIST CSF/RMF、ISO 27001、CIS Benchmarks；概念：纵深防御（Defense in Depth）、零信任（Zero Trust）、Runbook 运维手册
安全合规与管理
等级保护 2.0 实施、ISO 27001 体系建设、GDPR 合规要求、安全审计流程、风险评估方法论（NIST/OCTAVE）、安全策略编写
📚 推荐资源
《网络安全应急响应技术实战指南》、《CISSP官方学习指南》、ATT&CK 官方知识库、Elastic Security 官方文档、安全牛课堂
🎯 阶段目标
能独立搭建企业级安全监控体系、具备应急响应全流程处置能力、能编写安全策略和合规文档、理解蓝队防御体系全貌
📜 建议认证
CISSP / CISP / CySA+ / GCIH（GIAC应急响应）/ 等保测评师
阶段五

🌐 全栈开发能力（2-3年并行）

前端开发
HTML/CSS/JavaScript 扎实基础、React 或 Vue 框架（推荐 Vue3）、Tailwind CSS、安全工具平台前端开发、数据可视化（ECharts/D3.js）
后端开发
Python（Flask/FastAPI/Django）、Go 语言（高性能工具开发）、C++（底层安全工具）、JavaScript/Node.js、RESTful API 设计、GraphQL、数据库（MySQL/PostgreSQL/Redis/MongoDB）、消息队列（RabbitMQ/Kafka）
安全工具开发
自研漏洞扫描器（分布式架构）、自动化渗透框架、漏洞管理平台（资产管理+漏洞跟踪）、安全编排自动化（SOAR）、威胁情报平台开发、SIEM 与 SOAR 集成开发
云安全
AWS/GCP/Azure/阿里云安全配置与审计、云服务模型（SaaS/PaaS/IaaS）、云部署模型（公有云/私有云/混合云）、云原生安全（CSPM/CWPP）、Serverless 安全、IAM 策略与最小权限、云上攻防实战、Infrastructure as Code 安全
云存储安全
S3/OSS 存储桶安全配置、Dropbox/Box/OneDrive/Google Drive/iCloud 企业安全管理、云存储访问控制与加密、数据分类与敏感数据保护、云与本地环境安全差异
基础设施即代码
Docker 容器化部署、Kubernetes 集群管理、Ansible 自动化运维、CI/CD 流水线（GitHub Actions/Jenkins）、GitOps 工作流
安全平台架构
微服务架构设计、分布式系统基础、高可用与负载均衡、API 网关安全、OAuth2.0/JWT 认证体系、安全中台建设
📚 推荐资源
《Go语言编程》、《Flask Web开发实战》、Vue3 官方文档、Docker 官方文档、各云厂商安全白皮书
🎯 阶段目标
能独立开发完整的安全工具平台（前后端）、具备云环境安全评估能力、能用 Go/Python 开发高性能安全工具、有开源安全项目
📜 建议认证
AWS Security Specialty / CCSP / CKA（Kubernetes管理员）/ ACP 云安全
阶段六

🔬 高级专项方向（3-5年）

逆向工程与恶意软件分析
IDA Pro 高级使用（脚本/插件开发）、Ghidra 反编译分析、x86/x64/ARM 汇编语言、PE/ELF 文件格式、恶意样本静态分析、动态调试（x64dbg/GDB）、沙箱技术（Cuckoo/Any.Run）
二进制安全
栈溢出与堆溢出利用、Use-After-Free、ROP/JOP 链构造、Fuzzing 技术（AFL/LibFuzzer/Honggfuzz）、漏洞利用开发（Shellcode编写）、保护机制绕过（ASLR/DEP/Canary/CFI）
AI + 安全
机器学习检测恶意流量/恶意软件、深度学习异常检测、AI 辅助漏洞挖掘与代码审计、对抗样本攻防、LLM 安全（Prompt注入/越狱/数据泄露）、AI 模型安全评估
区块链安全

IoT 与工控安全

高级威胁研究
